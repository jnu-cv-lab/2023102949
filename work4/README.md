# 图像下采样与抗混叠实验
## 一、实验目的
1. 观察图像直接下采样产生的混叠现象，理解混叠产生的原理。
2. 掌握高斯低通滤波的抗混叠作用，对比直接下采样与先滤波后下采样的效果差异。
3. 验证高斯滤波标准差经验公式 σ = 0.45M 的有效性。
4. 实现基于梯度的自适应下采样，对比全局固定滤波与自适应滤波的优劣。
5. 结合FFT频谱，从频域角度分析混叠与抗混叠机制。

## 二、实验环境
- 运行环境：WSL Ubuntu + VS Code
- 编程语言：Python
- 依赖库：numpy、opencv-python、matplotlib、scipy

## 三、实验原理
1. **图像下采样**
每隔M个像素采样压缩，直接下采样不满足奈奎斯特采样定理，高频分量发生频率折叠，产生莫尔纹、畸变等混叠伪影。

2. **抗混叠滤波**
下采样前先进行高斯低通滤波，滤除高频分量，限制图像最高频率，再完成下采样，有效消除混叠。

3. **高斯σ经验公式**
下采样倍数为M，最优高斯标准差：
σ = 0.45 × M
σ过小混叠残留，σ过大图像过度模糊。

4. **自适应下采样**
利用Sobel算子计算图像梯度，纹理区域梯度大采用大σ强化滤波，平坦区域梯度小采用小σ保留细节，平衡去混叠与清晰度。

## 四、实验内容
1. 生成棋盘格、Chirp调频测试图像；
2. 实现倍数M=4直接下采样，观察空间混叠与频谱混叠；
3. 采用高斯滤波预处理，完成抗混叠下采样对比实验；
4. 设置多组不同σ参数，验证最优滤波系数；
5. 设计梯度自适应高斯滤波，实现自适应下采样并做误差对比。

## 五、实验结果
### 1. 棋盘格与Chirp图像下采样对比
![part1_result](part1_result.png)

直接下采样后，棋盘格出现明显莫尔混叠纹理，Chirp高频区域严重畸变；
经过高斯抗混叠滤波后，混叠伪影完全消除，FFT频谱无频率重叠，成像正常。

### 2. 不同σ参数对比实验
![part2_sigma_result](part2_sigma_result.png)

- σ=0.5、1.0：滤波不足，高频残留，仍存在混叠；
- σ=2.0、4.0：滤波过度，图像细节丢失、画面模糊严重。

### 3. 理论最优σ验证
![part2_opt_sigma_result](part2_opt_sigma_result.png)

固定下采样倍数M=4，计算得最优 σ=1.80。
该参数下既能完全抑制高频混叠，又最大程度保留图像细节，符合理论公式。

### 4. 自适应下采样实验结果
![part3_adaptive_result](part3_adaptive_result.png)

全局统一滤波存在缺陷：平滑区过度模糊、纹理区去混叠不足；
自适应下采样根据局部纹理动态调整σ，整体误差更小，细节保留更好，综合效果更优。

## 六、实验分析
1. 高频丰富的图像直接下采样必然产生严重混叠，破坏图像视觉质量；
2. 高斯低通滤波是简单高效的抗混叠手段，是图像下采样必不可少的预处理步骤；
3. σ=0.45M 经验公式实用性强，可作为图像压缩下采样的标准配置参数；
4. 全局单一滤波参数无法适配整张图像，基于梯度的自适应策略可以有效兼顾平坦区域与纹理区域；
5. FFT频谱分析可直观观测频率混叠现象，从原理上解释抗混叠滤波的作用。

## 七、实验总结
本次实验完成了图像直接下采样、高斯抗混叠下采样、σ参数对照、自适应下采样全部内容。
通过实验直观看到了混叠现象的产生与抑制过程，理解了采样定理与图像频域特性。
实验证明：图像压缩下采样必须配合低通抗混叠滤波，合理选择高斯核参数或采用自适应滤波，
可以在降低分辨率的同时，最大限度减少图像失真，保证图像处理质量。
